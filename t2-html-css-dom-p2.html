Whats all the hype about hyper responsive webpages

Designing webpages is not just for desktop computers now when you design a website you are designing a webpage that could be viewed and a varity of devices and screen sizes. Hyper responsive webpages adjust the content of a page automaticaly to be the best for the current device or screen that it is being viewed on which is very good for websites aiming for a wide audience.
